Lab 3: ASCII-risks (Asterisks)

DESCRIPTION:

In this lab, the user enters a value which is used as the height of the pattern which includes asterisks and numbers. If numbers less than 1 or any other input is given, an error message will be printed out prompting the user to enter another input. The asterisks and numbers on each line are separated by tabs, not spaces.

FILES:

Lab3.asm

This file contains the MIPS code.

INSTRUCTIONS:

To run the program, MARS is required.
